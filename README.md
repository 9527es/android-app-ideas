# android-app-ideas
 一个用于记录安卓系统自定义优化方向的清单项目，适合安卓爱好者和入门开发者参考。
 ## 项目简介
 本项目整理了安卓设备自定义过程中常见的功能需求与探索方向，聚焦于**解除系统限制**、**提升设备自由度**，帮助安卓用户挖掘设备潜力。
 ## 功能清单
 | 序号 | 功能方向 | 探索目标 |
 | :--- | :--- | :--- |
 | 1 | 应用安装限制解除 | 突破官方应用商店限制，安装第三方APK |
 | 2 | 系统图标美化 | 替换原生启动器图标，适配多分辨率屏幕 |
 | 3 | 后台应用保活 | 配置电池优化白名单，避免应用被后台查杀 |
 | 4 | 本地文件管理 | 获取完整文件访问权限，自由管理系统目录 |
